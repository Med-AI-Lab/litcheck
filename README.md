# litcheck
Alignment of hypothesis and existing literature

## How to run
To run the code, you'll need to provide openrouter key with `/.env` file:
```
OPENROUTER_API_KEY=sk-or...
```

Install the repository as a library:
```
!pip install git+https://github.com/Med-AI-Lab/litcheck.git
```
